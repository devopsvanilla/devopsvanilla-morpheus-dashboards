# Environment Count Widget - Guia de Configuração

## Visão Geral

Apresenta uma contagem consolidada de recursos do ambiente (ex.: instâncias, clouds, grupos) para visão rápida de escala.

## Código

- Componente: `morpheus-home-dashboard-plugin/src/assets/js/environment-count-widget.jsx`

## Pré-requisitos

- Morpheus 6.0+
- Plugin instalado
- Permissões: geralmente `provisioning` e/ou `infrastructure`

## Como adicionar

- Personalize o dashboard e inclua "Environment Count".

## Parâmetros

- Sem parâmetros obrigatórios.

## Dicas

- Útil como indicador geral de crescimento e capacidade do ambiente.
