# Instance Count by Cloud per Day Widget - Guia de Configuração

## Visão Geral

Apresenta a evolução diária da contagem de instâncias por cloud, permitindo analisar tendências e picos.

## Código

- Componente: `morpheus-home-dashboard-plugin/src/assets/js/instances/instance-count-cloud-day-widget.jsx`

## Pré-requisitos

- Morpheus 6.0+
- Plugin de Dashboard instalado
- Permissões: `provisioning`

## Como adicionar

- Adicione "Instance Count Cloud Day" na seção de Instâncias durante a personalização.

## Parâmetros

- Sem parâmetros obrigatórios.

## Dicas

- Útil para capacity planning e análise de crescimento.
