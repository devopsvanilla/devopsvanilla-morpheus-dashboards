# Current Alarms Widget - Guia de Configuração

## Visão Geral

Lista alarmes ativos ou recentes relacionados ao ambiente monitorado.

## Código

- Componente: `morpheus-home-dashboard-plugin/src/assets/js/health/current-alarms-widget.jsx`

## Pré-requisitos

- Morpheus 6.0+
- Plugin de Dashboard instalado
- Permissões: `infrastructure` ou equivalentes de monitoramento

## Como adicionar

- Personalize o dashboard e escolha "Current Alarms".

## Parâmetros

- Não há parâmetros obrigatórios.

## Dicas

- Use em conjunto com o widget de saúde para correlação rápida de eventos.
