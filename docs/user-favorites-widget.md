# User Favorites Widget - Guia de Configuração

## Visão Geral

Exibe os itens favoritos do usuário, facilitando o acesso rápido a recursos marcados.

## Código

- Componente: `morpheus-home-dashboard-plugin/src/assets/js/user-favorites-widget.jsx`

## Pré-requisitos

- Morpheus 6.0+
- Plugin instalado
- Permissões: dependem dos recursos favoritados

## Como adicionar

- Na personalização do dashboard, adicione "User Favorites".

## Parâmetros

- Sem parâmetros obrigatórios.

## Dicas

- Incentive usuários a marcar favoritos para aumentar produtividade.
