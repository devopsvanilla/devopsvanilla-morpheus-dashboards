# Current Health Widget - Guia de Configuração

## Visão Geral

Mostra indicadores de saúde atuais do appliance/ambiente (ex.: CPU, memória, storage, database, queue, search).

## Código

- Componente: `morpheus-home-dashboard-plugin/src/assets/js/health/current-health-widget.jsx`

## Pré-requisitos

- Morpheus 6.0+
- Plugin instalado
- Permissões: `infrastructure` ou `health`

## Como adicionar

- Nas preferências do dashboard, adicione "Current Health".

## Parâmetros

- Sem parâmetros obrigatórios.

## Dicas

- Ideal para visão geral do estado da plataforma.
