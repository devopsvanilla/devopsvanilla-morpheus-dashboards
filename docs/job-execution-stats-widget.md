# Job Execution Stats Widget - Guia de Configuração

## Visão Geral

Exibe estatísticas de execução de jobs (sucesso, falha, em andamento) para monitoramento operacional.

## Código

- Componente: `morpheus-home-dashboard-plugin/src/assets/js/jobs/job-execution-stats-widget.jsx`

## Pré-requisitos

- Morpheus 6.0+
- Plugin instalado
- Permissões: `job-executions`

## Como adicionar

- Personalize seu dashboard e inclua "Job Execution Stats".

## Parâmetros

- Sem parâmetros obrigatórios.

## Dicas

- Correlacione com falhas de tasks ou workflows para investigação.
