# Workflow Executions Over Time Widget - Guia de Configuração

## Visão Geral

Mostra a evolução temporal das execuções de workflows para acompanhamento de automações.

## Código

- Componente: `morpheus-home-dashboard-plugin/src/assets/js/workflow-executions-over-time-widget.jsx`

## Pré-requisitos

- Morpheus 6.0+
- Plugin instalado
- Permissões: `tasks` ou específicas de workflows

## Como adicionar

- Personalize o dashboard e adicione "Workflow Executions Over Time".

## Parâmetros

- Sem parâmetros obrigatórios.

## Dicas

- Analise junto a task/job stats para visão end-to-end de automações.
