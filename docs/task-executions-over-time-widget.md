# Task Executions Over Time Widget - Guia de Configuração

## Visão Geral

Apresenta a evolução temporal do volume de execuções de tasks para análise de tendências.

## Código

- Componente: `morpheus-home-dashboard-plugin/src/assets/js/task-executions-over-time-widget.jsx`

## Pré-requisitos

- Morpheus 6.0+
- Plugin instalado
- Permissões: `tasks`

## Como adicionar

- Nas preferências do dashboard, adicione "Task Executions Over Time".

## Parâmetros

- Sem parâmetros obrigatórios.

## Dicas

- Combine com "Task Failures" para correlacionar picos com falhas.
