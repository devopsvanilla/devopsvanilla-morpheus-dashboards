# Task Execution Stats Widget - Guia de Configuração

## Visão Geral

Exibe estatísticas de execução de tasks (sucesso, falha, etc.) oferecendo visão operacional consolidada.

## Código

- Componente: `morpheus-home-dashboard-plugin/src/assets/js/tasks/task-execution-stats-widget.jsx`

## Pré-requisitos

- Morpheus 6.0+
- Plugin instalado
- Permissões: `tasks`

## Como adicionar

- Personalize o dashboard e adicione "Task Execution Stats".

## Parâmetros

- Sem parâmetros obrigatórios.

## Dicas

- Correlacione com Job Execution Stats e Log Widgets para investigar falhas.
