# Log Trends Widget - Guia de Configuração

## Visão Geral

Apresenta tendências/seriados temporais de logs para identificar picos e padrões.

## Código

- Componente: `morpheus-home-dashboard-plugin/src/assets/js/logs/log-trends-widget.jsx`

## Pré-requisitos

- Morpheus 6.0+
- Plugin instalado
- Permissões: `logs`

## Como adicionar

- Nas preferências do dashboard, adicione "Log Trends".

## Parâmetros

- Sem parâmetros obrigatórios.

## Dicas

- Útil em troubleshooting e detecção de anomalias.
