# Group Workload Count Widget - Guia de Configuração

## Visão Geral

Exibe a contagem de workloads por grupo, auxiliando na visão por organização/projeto.

## Código

- Componente: `morpheus-home-dashboard-plugin/src/assets/js/groups/group-workload-count-widget.jsx`

## Pré-requisitos

- Morpheus 6.0+
- Plugin instalado
- Permissões: `infrastructure` ou correlatas

## Como adicionar

- Personalize o dashboard do usuário e adicione "Group Workload Count".

## Parâmetros

- Sem parâmetros específicos.

## Dicas

- Combine com filtros por grupo na UI.
