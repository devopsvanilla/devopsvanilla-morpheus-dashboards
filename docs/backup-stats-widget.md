# Backup Stats Widget - Guia de Configuração

## Visão Geral

O widget Backup Stats apresenta estatísticas resumidas de backups no seu dashboard.

## Onde está no código

- Componente: `morpheus-home-dashboard-plugin/src/assets/js/backups/backup-stats-widget.jsx`
- Registro: `MorpheusHomeDashboardPlugin.groovy` (providers do dashboard)

## Pré-requisitos

1. Morpheus 6.0+
2. Plugin instalado e carregado
3. Permissões adequadas (ex.: `backups`)

## Como adicionar ao Dashboard

- Personalize seu dashboard na UI e adicione "Backup Stats" na categoria de backups.

## Parâmetros de Configuração

- Sem parâmetros obrigatórios específicos.

## Dicas

- Verifique a conectividade com as fontes de backup e permissões do usuário.
