# Instance Count Widget - Guia de Configuração

## Visão Geral

Apresenta a contagem total de instâncias por status para rápida visibilidade do parque de instâncias.

## Código

- Componente: `morpheus-home-dashboard-plugin/src/assets/js/instances/instance-count-widget.jsx`

## Pré-requisitos

- Morpheus 6.0+
- Plugin instalado
- Permissões: `provisioning`

## Como adicionar

- Adicione "Instance Count" na seção de Instâncias ao personalizar seu dashboard.

## Parâmetros

- Sem parâmetros específicos.

## Dicas

- Útil para acompanhamento de crescimento e status operacionais.
