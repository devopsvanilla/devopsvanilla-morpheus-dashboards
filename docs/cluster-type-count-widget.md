# Cluster Type Count Widget - Guia de Configuração

## Visão Geral

Apresenta a quantidade de clusters por tipo, mostrando a distribuição de tecnologias ou perfis.

## Código

- Componente: `morpheus-home-dashboard-plugin/src/assets/js/clusters/cluster-type-count-widget.jsx`

## Pré-requisitos

- Morpheus 6.0+
- Plugin carregado
- Permissões: `infrastructure`

## Adição ao Dashboard

- Personalize seu dashboard e adicione "Cluster Type Count".

## Parâmetros

- Sem parâmetros customizáveis.

## Dicas

- Útil para inventário e governança de padrões.
