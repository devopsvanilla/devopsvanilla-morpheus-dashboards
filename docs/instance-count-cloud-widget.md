# Instance Count by Cloud Widget - Guia de Configuração

## Visão Geral

Mostra a contagem de instâncias por cloud, auxiliando na alocação e governança multi-cloud.

## Código

- Componente: `morpheus-home-dashboard-plugin/src/assets/js/instances/instance-count-cloud-widget.jsx`

## Pré-requisitos

- Morpheus 6.0+
- Plugin instalado
- Permissões: `provisioning`

## Como adicionar

- Na personalização do dashboard, adicione "Instance Count by Cloud".

## Parâmetros

- Sem parâmetros obrigatórios.

## Dicas

- Combine com filtros por grupo ou tag para análises mais específicas.
