# Task Failures Widget - Guia de Configuração

## Visão Geral

Foca em falhas de execução de tasks, permitindo identificar e priorizar correções.

## Código

- Componente: `morpheus-home-dashboard-plugin/src/assets/js/tasks/task-failures.jsx`

## Pré-requisitos

- Morpheus 6.0+
- Plugin instalado
- Permissões: `tasks`

## Como adicionar

- Personalize seu dashboard e adicione "Task Failures".

## Parâmetros

- Sem parâmetros obrigatórios.

## Dicas

- Use em conjunto com logs para investigação detalhada.
