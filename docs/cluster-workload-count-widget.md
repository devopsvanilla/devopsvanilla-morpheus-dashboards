# Cluster Workload Count Widget - Guia de Configuração

## Visão Geral

Mostra a contagem de workloads por cluster para acompanhar distribuição de carga.

## Código

- Componente: `morpheus-home-dashboard-plugin/src/assets/js/clusters/cluster-workload-count-widget.jsx`

## Pré-requisitos

- Morpheus 6.0+
- Plugin de Dashboard instalado
- Permissões: `infrastructure`

## Como adicionar

- Nas preferências do dashboard, adicione "Cluster Workload Count".

## Parâmetros

- Sem parâmetros obrigatórios.

## Dicas

- Relacione com capacidade para identificar gargalos.
