# Cloud Workload Count Widget - Guia de Configuração

## Visão Geral

Mostra a quantidade de workloads distribuídas por cloud.

## Onde está no código

- Componente: `morpheus-home-dashboard-plugin/src/assets/js/clouds/cloud-workload-count-widget.jsx`
- Registro: `MorpheusHomeDashboardPlugin.groovy`

## Pré-requisitos

- Morpheus 6.0+
- Plugin de Dashboard instalado
- Permissões: `infrastructure`

## Como adicionar ao Dashboard

- Personalize o dashboard do usuário e adicione "Cloud Workload Count" na seção de clouds.

## Parâmetros de Configuração

- Sem configurações obrigatórias específicas.

## Dicas

- Útil para identificar hotspots de utilização por cloud.
