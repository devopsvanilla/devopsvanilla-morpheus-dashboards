# Settings Widget - Guia de Configuração

## Visão Geral

Widget de configurações do dashboard, auxiliando em ajustes e preferências do usuário.

## Código

- Componente: `morpheus-home-dashboard-plugin/src/assets/js/settings-widget.jsx`

## Pré-requisitos

- Morpheus 6.0+
- Plugin instalado

## Como adicionar

- Personalize seu dashboard e adicione "Settings".

## Parâmetros

- Sem parâmetros obrigatórios.

## Dicas

- Use para acessar rapidamente preferências de exibição e ajustes de layout.
