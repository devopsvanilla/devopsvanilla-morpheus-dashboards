# Log Count Widget - Guia de Configuração

## Visão Geral

Mostra contagens de logs por status/categoria para visão rápida de volume e severidade.

## Código

- Componente: `morpheus-home-dashboard-plugin/src/assets/js/logs/log-count-widget.jsx`

## Pré-requisitos

- Morpheus 6.0+
- Plugin instalado
- Permissões: `logs`

## Como adicionar

- No dashboard do usuário, adicione "Log Count".

## Parâmetros

- Sem parâmetros obrigatórios.

## Dicas

- Combine com Log Trends para contexto temporal.
