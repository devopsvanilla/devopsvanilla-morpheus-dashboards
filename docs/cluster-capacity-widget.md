# Cluster Capacity Widget - Guia de Configuração

## Visão Geral

Exibe métricas de capacidade dos clusters (ex.: uso de recursos) para ajudar no planejamento e operação.

## Onde está no código

- Componente: `morpheus-home-dashboard-plugin/src/assets/js/clusters/cluster-capacity-widget.jsx`

## Pré-requisitos

- Morpheus 6.0+
- Plugin instalado
- Permissões: `infrastructure`

## Como adicionar

- Personalize seu dashboard e escolha "Cluster Capacity" na categoria de clusters.

## Parâmetros

- Sem parâmetros obrigatórios; utiliza contexto do usuário.

## Dicas

- Combine com cluster workload para visão completa de uso vs carga.
