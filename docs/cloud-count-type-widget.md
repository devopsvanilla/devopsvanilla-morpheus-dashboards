# Cloud Count by Type Widget - Guia de Configuração

## Visão Geral

Apresenta a contagem de clouds por tipo, ajudando a entender a distribuição do ambiente.

## Onde está no código

- Componente: `morpheus-home-dashboard-plugin/src/assets/js/clouds/cloud-count-type-widget.jsx`
- Registro: `MorpheusHomeDashboardPlugin.groovy`

## Pré-requisitos

- Morpheus 6.0+
- Plugin de Dashboard instalado
- Permissões: `infrastructure`

## Como adicionar ao Dashboard

- Personalize o dashboard do usuário e adicione "Cloud Count by Type" na seção de clouds.

## Parâmetros de Configuração

- Sem configurações obrigatórias específicas; segue escopo do usuário.

## Dicas

- Combine com filtros globais para análises por grupo/cloud.
